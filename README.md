# Constraint-Based Waste Management System (CSP)

## Problem Statement
This project designs an intelligent waste collection system for a city using
Constraint Satisfaction Problem (CSP) techniques to optimize routes and schedules
based on bin fill levels, vehicle capacity, and distance.

## Approach
- Model waste bins, vehicles, and routes as CSP variables
- Apply constraints such as capacity limits, priority bins, and distance
- Generate optimized collection routes

## Technologies Used
- Python
- CSP / AI Planning
- Basic data structures

## Results
- Optimized routes generated
- Reduced travel distance and improved waste collection efficiency

## How to Run
1. Clone the repository
2. Run `main.py`
